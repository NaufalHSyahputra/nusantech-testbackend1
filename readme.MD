## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Run index.html in browser

Or Run in terminal using node (Requirement: NodeJS Already Installed)

```bash
  node maze1.js
```
